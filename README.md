# Logic Programming Exercises

This project contains solutions to a set of 22 logic programming exercises based on the list provided by DIO:  
https://www.dio.me/articles/lista-de-exercicios-para-treinar-logica-de-programacao

The goal of this repository is to practice fundamental programming concepts such as variables, conditionals, loops, arrays, functions, and basic input/output using Java.

---

## Project Structure

- Each exercise is implemented as a separate class.
- A main menu allows selecting and executing any exercise.
- All solutions focus on clarity and beginner-friendly logic.

---

## Technologies Used

- Java 21
- IntelliJ IDEA
